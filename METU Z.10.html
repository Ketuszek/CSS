1. Wymień i opisz dwie części selektora

    Selektor składa się z dwóch części:
    1. Selektor właściwy – określa element HTML (np. p, h1, body).
    2. Deklaracja – określa styl jaki ma być zastosowany do elementu.

  2. Znaczenie nawiasów klamrowych { }

    Nawiasy klamrowe { } obejmują część deklaracji CSS i pokazują, które style dotyczą danego selektora.

  3. Co to są cechy?

    Cechy (np. color, font-size) określają co chcemy zmienić w wyglądzie elementu.

  4. Co to są wartości?

    Wartości to konkretne ustawienia cech, np. red, blue, 20px.

  5. Kiedy stosujemy średnik (;)?

    Średnik ; stosujemy po każdej deklaracji CSS, aby oddzielić kolejne cechy. Nie jest konieczny tylko przy ostatniej cesze w bloku.
