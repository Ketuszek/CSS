Definicja CSS

CSS (Cascading Style Sheets) to język służący do opisu wyglądu dokumentów HTML. Pozwala na oddzielenie treści strony od jej prezentacji, czyli określenie kolorów, czcionek, układu elementów oraz ich stylu.

Skrót CSS pochodzi z języka angielskiego i oznacza: Cascading Style Sheets, czyli Kaskadowe Arkusze Stylów.

Dzięki CSS możliwe jest tworzenie estetycznych i czytelnych stron internetowych oraz łatwe zarządzanie ich wyglądem poprzez jeden plik stylów.
